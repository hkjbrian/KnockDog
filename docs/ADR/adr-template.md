# ADR 템플릿

새 ADR을 만들 때 **이 파일을 복사**해 `docs/ADR/NNNN-슬러그.md`로 저장하고 아래 항목을 채운다.
헤더의 시간정보는 `생성: YYYY-MM-DD HH:MM` 형식(KST)으로 **정확한 시각**을 넣는다.
섹션 순서는 반드시 **맥락 → 검토한 후보 → 결정 → 결과** (후보를 먼저 제시하고, 그 뒤에 결정이 나오도록).

- **후보 A**: <설명 / 장단점>
- **후보 B**: <설명 / 장단점>
